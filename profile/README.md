## 💡 Overview

<a href="#lunacy">Lunacy</a> is a free professional vector design application for Mac from Icons8 — providing a complete UI design, illustration, and graphic design tool with built-in access to millions of assets including stock photos, icons, illustrations, and avatars, entirely at no cost and without subscription.

The <a href="#lunacy">built-in Icons8 asset library</a> is Lunacy's most distinctive advantage — directly within the design tool, designers access millions of icons in dozens of styles, high-quality stock photos, illustrations, avatars, and background textures without leaving the application. Placing a photo in a mockup or finding an icon for a UI element requires only a search within Lunacy's built-in asset panel. The <a href="#lunacy">Sketch file compatibility</a> opens and edits Sketch files directly — the de-facto standard format for UI design work means Lunacy can participate in existing Sketch-based workflows without file conversion.

The <a href="#lunacy">component and symbol system</a> creates reusable design elements — buttons, navigation bars, form fields, and any other repeated UI element created once and used throughout the design, updated globally when the master component changes. The <a href="#lunacy">auto layout capability</a> builds responsive design components that adapt to content changes — text boxes that expand to fit their content, button padding that maintains spacing as text changes.</p>

<a href="#lunacy">AI design tools</a> include background removal, image upscaling, and smart suggestions — AI-powered operations that reduce manual design effort for common image preparation tasks. <a href="#lunacy">Real-time collaboration</a> allows multiple designers to work on the same file simultaneously — design review and team iteration without version conflict.

<a href="#lunacy">Text and typography tools</a> provide complete control over typeface, size, weight, and spacing — with Google Fonts access built in for typographic variety without font installation overhead. <a href="#lunacy">Export options</a> produce assets in PNG, SVG, PDF, and other formats for developer handoff, print, and web-optimized delivery from the same design.</p>
